# Data DVC Demo
Data version control using DVC
 
## How to setup DVC project

```sh
$ git clone <repo-url>
$ cd <repo-name>
$ dvc init
$ git status
$ git add .
$ git commit -m "Initial commit"
```
### Add git repository to DVC
```sh
$ dvc remote add <remote-name> <remote-url>
$ dvc remote add <remote-name> <remote-url>
```
